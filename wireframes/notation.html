Verwendete Software

Die Wireframes wurden direkt mit HTML & CSS in Visual Studio Code erstellt – ohne Adobe XD oder externe Tools. Vorteile:

  • 1:1-Übereinstimmung mit der späteren Implementierung (gleiche Breakpoints)
  • Versionierbar via Git
  • Direkt im Browser auf verschiedenen Auflösungen prüfbar

Die Screenshots für die Word-Abgabe wurden im Browser bei 1200 px (Desktop) bzw. 380 px (Mobile) erzeugt.
